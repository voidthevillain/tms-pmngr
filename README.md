# TEAMS - Profile Manager
Teams tool for managing custom profiles.

### Prerequisites
WindowsOS w/ PowerShell 5.1

### Install
Download the bin\TEAMS - Profile Manager.exe file

### Run
TEAMS - Profile Manager.exe

Or PowerShell: .\tms-ProfileManager.ps1
